## 📝 Description
Nom appli : Fileshare

Ceci est un projet personnel qui permet à un adminstrateur de partager des fichiers via qrcode permettant l' accès aux fichiers des personnes ayant le qrcode.

## ✅ Prérequis

- Node.js et npm installés
- PostgreSQL installé
- Git installé

## 🎯 Objectifs

Ce projet vise à consolider les bases en backend, structurer un projet complet (frontend + backend + BD) et appliquer des notions de gestion de projet et découvrir des pratiques DevOps.

## 🧰 Outils

- Frontend : React
- Backend : Node.js + Express
- BD : PostgreSQL

ℹ️ **PS** : Les versions ont été choisies pour être récentes mais stables.

## Installation
1. Récupérer le repo:

```
git clone url
cd dir
```

2. 🛠️ Installer les dépendances

⚙️ Pour le côté backend :
```
cd backend
npm install
```
🎨 Pour le côté frontend :

```
cd frontend
npm install
```
3. Ajouter le fichier `.env`

Renommer le fichier `.env.exemple` à la racine du projet et changer les valeurs selon vos configurations locales.

4. 🚀 Lancement

Dans frontend/
```
npm run dev
```

Et dans backend/
```
npm start
```
